# ZK Project

Create and manage project notes in the Zettelkasten.

## Workflow

1. Collect from user: title, objective, tasks, priority, deadline
2. `zk_find_connections` or `zk_list` — find related permanent/literature notes
3. Present project plan for confirmation
4. `zk_project` tool — create in 5-Projects/
5. Suggest linking related notes

## Rules

- All content in Ukrainian
- Each project must have a clear objective
- Link to relevant permanent notes for context
- Priority: low/medium/high
