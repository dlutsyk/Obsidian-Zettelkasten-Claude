# ZK Daily

Morning briefing for the Zettelkasten vault.

## Workflow

1. `zk_unprocessed` — sort by age, show urgency:
   - 🔴 >14 days = critical — needs immediate attention
   - 🟡 >7 days = warning — should be processed soon
   - ⚪ ≤7 days = normal
2. `zk_orphans` for folder=3-Permanent — disconnected ideas
3. `zk_cluster_detect` — emerging themes without MOCs
4. `zk_status` — vault stats

Present as concise Ukrainian briefing with action suggestions. Prioritize critical items first.
